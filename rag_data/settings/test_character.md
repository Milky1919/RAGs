# キャラクター設定：氷室玲

## 基本情報

氷室玲（ひむろ れい）は本作のメインヒロイン。
高校2年生、17歳。クラス委員長を務める優等生。

## 外見

黒髪のロングヘアに切れ長の瞳。
いつも制服をきっちり着こなし、隙がない印象を与える。
しかし、ふとした瞬間に見せる表情は柔らかく、その落差に気づく者は少ない。

## 性格

「氷の女王」と呼ばれるほどクールで、人との距離を置く。
実は読書好きで、放課後の図書室に一人でいることが多い。
主人公の佐藤裕馬だけには、なぜか素の表情を見せてしまう。

## 口癖

「別に、あなたに話すことは何もない」
「……なんでもない」

## 秘密

幼い頃の経験から、人を信じることが怖くなっている。
図書室で泣いているところを裕馬に見られたことが、物語の転換点となる。
